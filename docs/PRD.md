# PRD: ChallengePick

## 목표
요즘 핫하고 바이럴한 아이돌 챌린지/댄스 영상을 자동으로 찾아 추천해주는 웹 앱.

## 사용자
- 아이돌 챌린지 트렌드를 빠르게 따라잡고 싶은 K-pop 팬
- 챌린지 영상을 찍거나 큐레이션하려는 크리에이터

## 핵심 기능
1. YouTube에서 아이돌 챌린지/댄스챌린지 키워드로 영상 검색
2. 최근(기본 30일 이내) 업로드된 영상 중 조회수 높은 순으로 정렬
3. 썸네일·제목·채널·조회수·업로드일·바로가기 링크를 카드 그리드로 추천 표시

## MVP 제외 사항
- 사용자 로그인/계정
- 즐겨찾기 저장, 개인화 추천(히스토리 기반)
- 영상 재생(임베드 플레이어) — 외부 링크로 대체
- 무한 스크롤/페이지네이션 — 상위 N개만 표시
- DB/영구 저장소 — 매 요청 시 실시간 조회

## 디자인
- 다크모드 고정, 미니멀
- 무채색 베이스 + 포인트 컬러 1가지(핑크 계열, K-pop 무드)
- 카드 그리드 중심, 모바일 우선 반응형
